## NES Emulator
A NES Emulator in C++ using SDL2.

#### How to Run
From the root directory, run:  
`make all`  
Then run the created main binary with:  
`./main <ROM file>`

#### Controls
The 16 CHIP8 buttons are mapped as:  
`1` `2` `3` `4`  
`q` `w` `e` `r`  
`a` `s` `d` `f`  
`z` `x` `c` `v`  

The speed of the running ROM can also be changed with:  
`9`: Decrease speed.  
`0`: Increase speed.
